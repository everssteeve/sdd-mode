# INTENT-103-retention-30j

**Auteur** : Steeve
**Date** : 2026-04-20
**Statut** : active

---

## POURQUOI MAINTENANT

Rétention J+30 oscille entre 38 % et 52 % depuis 3 mois. Le board veut une
stabilité ≥ 55 % pour passer la phase de scale-up.

## POUR QUI

Cohorte utilisateurs activés (≥ 1 lien créé dans les 24 h).

## OBJECTIF

Atteindre 55 % de rétention J+30 stable sur 4 cohortes consécutives.

## CONTRAINTES

- RGESN (limiter notifications / push)
- Pas d'ajout de feature majeure — uniquement boucle réengagement existante

## CRITÈRE DE DRIFT

Si une cohorte tombe sous 45 % → drift, on stoppe les itérations et on
revoit la boucle d'engagement à partir des données comportementales.
