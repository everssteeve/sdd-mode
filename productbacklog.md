# Product Backlog — aiad-sdd

> Backlog actif pour faire d'`aiad-sdd` le **framework leader européen de développement basé sur l'intention** (langue française par défaut, ouverture EU).
>
> Convention : items priorisés du plus haut levier au plus faible. Chaque item livre une amélioration vérifiable par tests automatisés.
>
> Marquage : `[ ]` à faire · `[x]` fait · `[~]` en cours · `[!]` bloqué.

## Cap stratégique

- **Différenciation** : seul framework SDD multi-runtime qui combine cycle Intent→SPEC→Gate→Drift Lock + gouvernance EU Tier 1 (RGPD/RGAA/AI-ACT/RGESN) + traçabilité machine-vérifiable.
- **Marché cible primaire** : équipes produit/ingénierie francophones (FR/BE/CH/LU/QC) ; ouverture EU via `AGENTS.md` standard et runtimes multiples.
- **Promesse produit** : "Tu écris une intention, l'agent livre du code traçable, conforme et auditable — sans drift."
- **Standards qualité** : zero-dep runtime, tests automatisés, CI verte, idempotence stricte, sortie machine-lisible (JSON/SARIF) sur toutes les commandes.
